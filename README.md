# md.boidu.dev

A dead-simple, local-first minimal markdown editor with:

- Multi-view layout: Editor, Split, Preview (toggle with ⌘1/2/3)
- Full GFM support: Tables, task lists, code blocks with syntax highlighting
- Multiple projects: Stored in IndexedDB, auto-save with title extraction
- Draggable split pane: Persists ratio to localStorage
- Solar icons: Clean Linear style throughout
- Keyboard shortcuts: ⌘B sidebar, ⌘N new, ⌘/ help
